## Hypothesis 1 — "enforcement must survive the agent it constrains"

<!-- NOT IMPLEMENTED: frame the first bet as a testable hypothesis, then show the challenge
     and the resolution.
     BET: a guard only counts if the constrained agent cannot quietly turn it off. So the
          covenant must protect its OWN substrate — recursively.
     CHALLENGE: dogfooding a guard that isn't yet guarded is a contradiction. If we protect
          the tool axis but leave Bash open, the agent disables the covenant with `sed -i`.
          => both axes had to close BEFORE self-dogfooding turned on (2026-07-14).
     RESOLUTION + the structural payoff we found: the escape hatch is human-only by
          construction (arming it requires editing a file that is itself protected), so the
          agent cannot self-authorize a bypass. Report this as an engineering result.
     SCOPE HONESTY + forward signal (per decision 2026-07-17): today this human-only property
          is realized THROUGH ONE RUNTIME — Claude Code, where the valve is an env var armed
          only by editing `.claude/settings.json`, itself a protected path. That is an
          adapter-specific binding, not yet an agent-neutral guarantee. State plainly that
          the *principle* ("only a human, out of session, can arm the bypass") is what
          generalizes, and that each future adapter (Codex, pi, …) must supply its own
          human-only arming path grounded in that runtime's equivalent of "the settings
          file is itself a protected surface." Frame as a designed obligation of the adapter
          layer, consistent with the neutral-IR goal — not a gap we're hiding. -->


## Hypothesis 2 — "deterministic gates beat probabilistic checks — but only if you measure them"

<!-- NOT IMPLEMENTED: the second bet, and where v0.1 data challenged AND confirmed it.
     BET: exit-code / fail-closed judgment ("cannot judge = block") is a stronger guarantee
          than an LLM reviewer or a prompt-level rule.
     CONFIRMED: the milestone audit is the proof — the machine caught 5 real Bash-axis
          bypasses the humans missed after the backlog looked done. Determinism found what
          eyes didn't. (And then three review passes to converge the fix — tell it as
          process: each fix opened a new surface, the discipline caught each one.)
     CHALLENGED: determinism has a cost, and if you don't measure it you're flying blind.
          bypassed jumped 56 → 552 across the milestone. That number isn't hidden — it
          reordered the roadmap (TTL waiver pulled forward). This is GAP B, answered on
          ourselves: we measure what the enforcement layer costs, not just what the model
          outputs. -->
